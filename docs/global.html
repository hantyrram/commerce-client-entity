(constant) MODE

If no entity is prop the form is said to be in ADD mode, it means the form can be used in adding an entity. If there is an entity prop but the entity has no _id property, the form is in READ mode. If there is an entity prop AND the entity HAS _id property, the form is in EDIT mode.
